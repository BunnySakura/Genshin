## 使用方法
**1. 运行程序在弹出窗口选择原神安装目录,支持模糊匹配;**  
**2. 匹配完成自动重命名对应目录,并且生成一个GenshinDebug.bat文件于桌面.此后若反和谐失效,双击GenshinDebug.bat即可;**  

若不记得原神安装在哪个磁盘,支持各磁盘全盘搜索
